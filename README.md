# Java Assignment 3 - Student Information Processor

## 📌 Project Description
This Java program collects and processes student information using variables, data types, and conditional statements. It displays a formatted student profile including academic classification.

---

## 🚀 Features
- Accepts student details (name, index number, age, gender, department, level, GPA)
- Determines adult status
- Calculates academic classification:
  - First Class
  - Second Class Upper
  - Second Class Lower
  - Pass
- Displays formatted student profile

---

## 🛠 Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

---

## ▶️ How to Run
1. Open project in IntelliJ IDEA
2. Run `Main.java`
3. Enter required inputs in console
